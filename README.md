# Wildlife Monitoring System

A real-time wildlife detection and monitoring project using computer vision. This system detects animals from video feeds using deep learning models and can be used for wildlife surveillance and conservation.

## Features

- Real-time animal detection from video streams  
- Uses state-of-the-art object detection model (YOLOv5)  
- CNN model for refined classification  
- Image preprocessing using OpenCV  
- Easy integration with live or recorded videos  

## Technologies Used

- Python  
- YOLOv5 (object detection)  
- Convolutional Neural Network (CNN)  
- OpenCV (image/video processing)  
- PyTorch (deep learning framework)  
- Numpy, Matplotlib, and other supporting libraries
